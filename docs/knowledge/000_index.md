# Indice de Conocimientos Incrementales

Esta carpeta guarda conocimientos acumulados del proyecto. Cada nota se numera incrementalmente para conservar historia y contexto.

## Politica

- Las notas nuevas se agregan con el siguiente numero disponible.
- Si un conocimiento nuevo contradice uno anterior, no se debe editar silenciosamente la nota vieja.
- Primero se debe documentar la contradiccion propuesta y pedir aprobacion del usuario.
- Solo despues de la aprobacion se actualiza la nota anterior o se crea una nota nueva marcada como reemplazo.
- Si una nota queda reemplazada, debe indicarlo con una seccion `Estado`.
